MyBiz Action Reports

How to access Action Reports
  1. Access Action Reports

    ABO must click on the menu icon at the top left hand corner to navigate the different MyBiz sections. ABO must click on ‘Action Reports’ to proceed to view Action Reports section. ABO can also navigate directly by expanding the Action Reports section and selecting a specific report they want to view.

Action Reports Page:
Landing
  1. Navigate MyBiz

    ABO can click on the icons to access different sections of MyBiz.

  2. Action Reports

    In this section the ABO can view a summary of 4 Action Reports. ABO can see how many Downlines appear in each and by clicking on the report, ABO can proceed to the List View of that Report.

  3. Who’s New

    ABO can see an overview of their new Downlines and can view a more detailed list by clicking on this report.

  4. Who’s Close

    ABO can see an overview of their Downlines who are close to their next Bonus Level and can view a dynamic list by clicking on this report.

  5. Renewals

    ABO can see an overview of their Downlines who are due to renew soon can view a more detailed list by clicking on this report.

  6. Birthdays

    ABO can see an overview of their Downlines’ Birthdays and can view a more detailed list by clicking on this report.

Action Reports Page:
Who’s New Report
  1. Navigate

    ABO can switch between Action Reports by selecting a report from the dropdown list on the left. Clicking the icon on the right will open a panel to configure the report.

  2. Choose Year

    ABO can toggle between the current and previous year in this report by clicking on the left and right arrow icons.

  3. View

    ABO can select the left hand icon for a List View or the center icon to see a Tree View. ABO can customize report columns by selecting the icon on the right hand side.

    ***By default ABO will see the List View.

  4. Choose Month

    ABO must click on the Month to view a list of who was new in that month. ABO can jump to the Current month by clicking the ‘Curret Month’ button in the top right corner.

  5. Summary

    ABO can see a summary of how many Downlines were new this month in the small orange badges.

  6. List View

    ABO can see a more detailed list view of the report below. This will automaticallyupdate depending on the month selected.

Action Reports Page:
Who’s Close Report
  1. Navigate

    ABO can switch between Action Reports by selecting a report from the dropdown list on the left. Clicking the icon on the right will open a panel to configure the report.

  2. Select Percentage

    ABO can see which of their Downlines are close to their next Bonus Level by dragging the slider to select the percentage to the next bonus level they want to view in the report.

  3. View

    ABO can select the left hand icon for a List View or the center icon to see a Tree View. ABO can customize report columns by selecting the icon on the right hand side.

    ***By default ABO will see the List View.

  4. Apply

    To complete the configuration for this Action Report, ABO must click ‘Apply’ button on the right hand side.

  5. List View

    ABO can see a more detailed list view of the report below. This will automatically update the report when ABO clicks ‘Apply’ button.

Action Reports Page:
Renewals Report
  1. Navigate

    ABO can switch between Action Reports by selecting a report from the dropdown list on the left.

    ***There is no option to configure this report.

  2. Renewals

    ABO can see a list of the outstanding Renewals in their group. All ABOs that have not renewed in the current Renewal Period will show up on the list. Below them, ABOs in the Late Renewal Period will be displayed.

  3. View

    ABO can select the left hand icon for a List View or the center icon to see a Tree View. ABO can customize report columns by selecting the icon on the right hand side.

    ***By default ABO will see the List View.

  4. List View

    ABO can see a list view of the report below. ABOs are sorted by how urgent their Renewal Date is and then by Group Size.

Action Reports Page:
Birthdays Report
  1. Navigate

    ABO can switch between Action Reports by selecting a report from the dropdown list on the left.

    ***There is no option to configure this report.

  2. Choose Month

    ABO must select a month to view all the ABOs with Birthdays in that month.

  3. View

    ABO can select the left hand icon for a List View or the center icon to see a Tree View. ABO can customize report columns by selecting the icon on the right hand side.

    ***By default ABO will see the List View.

  4. List View

    ABO can see a list view of the report below. ABOs are sorted by how urgent their Renewal Date is and then by Group Size.

Action Reports Page:
Tree View
  1. Navigate

    ABO can switch between Action Reports by selecting a report from the dropdown list on the left.

  2. Zoom

    At first, ABO will see a zoomed out Tree View. ABO can use the scale on the left hand side or click on a specific Downline to zoom in on the Tree and take a closer look.

  3. View

    ABO can select the left hand icon for a List View or the center icon to see a Tree View. The Tree is filtered to highlight only the downlines that appear on the report. ABO can customize report by selecting the icon on the right hand side.

  4. Tree View

    In this section ABO can see a quick overview of all the Downlines that fit the Action Report they are viewing.

How to switch between reports
  1. Switch Reports

    ABO must click on the report dropdown and choose another report from the list to switch between Action Reports.
